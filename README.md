# Calculator Project

## Description
This project is a simple calculator implemented in C# using the .NET framework. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Square root
- clear
